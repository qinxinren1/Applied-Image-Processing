Code Organization

The project that you received consists of several files and folders:

File Description
src Directory contains all the c++ source files (both .cpp implementation and .h header files)
framework Contains the framework code (code that is shared between the different assignments)
CMakeLists.txt Tells the CMake build system how to compile/link the program

The src directory contains the only files that you have to interact with. The assignments are set up in such a way that you only have to modify one or two files each time. None of the assignments should require you to change the framework or CMakeLists.txt.
